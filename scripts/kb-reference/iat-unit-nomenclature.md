# IAT Unit Nomenclature (2022)

Internal reference for how Innovative Air Technologies (IAT) model numbers and
serial numbers are constructed, plus the current standard product lineup. Source:
IAT "Unit Nomenclature" sheet (2022) and IAT leadership. Use this to decode any
IAT model number a customer or ticket references.

## How to read an IAT model number

Worked example: **IAT-600REHC/IDP-450**

| Segment | Example | Meaning |
| --- | --- | --- |
| Manufacturer | `IAT` | Innovative Air Technologies. |
| Nominal CFM | `600` | Standard catalog airflow size (see the list below). |
| System Type | `R` | `R` = Rotor · `D` = Drum (OBS, obsolete) · `B` = Bed (OBS, obsolete) · `AHU` = Air Handling Unit. |
| Reactivation Type | `E` | How the desiccant wheel is regenerated: `E` = Electric · `S` = Steam · `G` = Gas · `HW` = Hot Water. |
| High Capacity | `HC` | Added **only** when a high-capacity desiccant wheel is required. Absent = standard wheel. |
| Integrated Dehumidification Package | `/IDP` | Added when the unit is a customized package with integrated components — e.g. Pre-Cooling, Post-Cooling, Heating, Filtration. |
| Actual CFM | `450` | Added **only** when the actual airflow differs from the nominal (catalog) CFM of that size. |

So `IAT-600REHC/IDP-450` = a nominal-600-CFM **R**otor unit, **E**lectric
reactivation, **H**igh-**C**apacity wheel, built as an **IDP** integrated package,
moving an **actual 450 CFM**.

### Nominal CFM catalog sizes
75, 150, 300, 600, 1,000, 1,500, 3,000, 5,000, 7,500, 10,000, 15,000, 20,000,
25,000, 30,000 CFM.

### Reactivation letters
- `E` — Electric
- `S` — Steam
- `G` — Gas
- `HW` — Hot Water

### `HC` — High Capacity
`HC` appearing after the `R` (system) and `E`/`S`/`G`/`HW` (reactivation)
designations means a **High-Capacity desiccant wheel** is fitted (e.g.
`IAT-7500RGHC-IDP`). No `HC` = standard-capacity wheel.

### `IDP` — Integrated Dehumidification Package
`/IDP` (or `-IDP`) marks a customized unit that integrates additional components.
Typical IDP options: Pre-Cooling, Post-Cooling, Heating, Filtration. **A number
after the `IDP` portion indicates the unit's actual airflow in CFM when it differs
from the nominal CFM of that size** — e.g. `IAT-600RG-IDP-1200` is a nominal-600
Rotor/Gas IDP that actually moves 1,200 CFM.

## Serial number format

Example serial: **93-3435**
- First part (`93`) — the **year of sale**.
- Second part (`3435`) — the **sequential order of sale (perpetual)**; it keeps
  counting up across years, it does not reset annually.

## Current standard product lineup (per IAT leadership)

### Compacts
Small packaged rotor units. Designated `REC` — Rotor / Electric reactivation /
**C**ompact.
- IAT-75REC
- IAT-150REC
- IAT-300REC
- IAT-600REC

### Rotors
Standard rotor dehumidifiers, sized by nominal CFM.
- IAT-600
- IAT-1000
- IAT-1500
- IAT-3000
- IAT-5000
- IAT-7500
- IAT-10000
- IAT-15000
- IAT-20000
- IAT-30000

### IDPs (Integrated Dehumidification Packages)
An IDP can be **any** of the Compact-series or Rotor-series units, but customized
with integrated components. Naming follows the same rules as above:
- `HC` after the `R` / reactivation letters = High Capacity.
- A number after the `IDP` portion = actual airflow in CFM, when different from the
  unit size's nominal CFM.

Example IDP models:
- IAT-75REC-IDP
- IAT-600RG-IDP-1200  (nominal 600, Rotor/Gas, actual 1,200 CFM)
- IAT-5000RS-IDP-4000 (nominal 5,000, Rotor/Steam, actual 4,000 CFM)
- IAT-7500RGHC-IDP    (nominal 7,500, Rotor/Gas, High-Capacity wheel)
